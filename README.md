# PH 142 Group Project

Collaborative repository for our PH 142 biostatistics group project.

## Team workflow

1. Pull before beginning work.
2. Edit only your assigned files.
3. Only one person edits group_project.Rmd at a time.
4. Commit with a descriptive message.
5. Pull again before pushing.
6. Push before ending a work session.

## Main files

- group_project.Rmd: cumulative final report
- dataset.csv: project dataset
- scripts/: individual analysis and exploratory scripts
- figures/: saved visualizations
